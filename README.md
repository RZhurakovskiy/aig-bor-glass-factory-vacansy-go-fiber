# aig-bor-glass-factory-vacansy-go-fiber

## Bootstrap admin

Пароль главного администратора больше не хранится в коде.

Если `hrautomotive_admin` еще не создан в пустой базе, задайте одноразово:

```bash
BOOTSTRAP_ADMIN_PASSWORD='your-strong-password'
```

При следующем запуске сервер создаст главного пользователя. После этого переменная больше не нужна.

`SESSION_SECRET` пакет теперь создает автоматически при установке в `/etc/glass-factory/glass-factory.env`. Если переменная не задана, сервер все равно сгенерирует временный секрет на текущий запуск, но после рестарта админ-сессии будут сброшены.

## Reverse proxy

Если приложение стоит за `nginx`, добавьте адреса доверенных прокси в `TRUSTED_PROXIES`, иначе метрики будут видеть IP самого прокси:

```bash
TRUSTED_PROXIES='127.0.0.1,::1'
```

Если `nginx` или балансировщик находится в docker-сети или на другом хосте, добавьте его IP или CIDR в эту переменную.
